# HMRC Procurement Spend Analytics

## Overview
End-to-end procurement spend analysis project built on real UK government spending data published by HM Revenue & Customs (HMRC). The project covers data ingestion, cleaning, SQL-based analysis, and an interactive Power BI dashboard.

## Tools Used
- **MySQL** — data storage, cleaning, and analytical queries
- **Power BI** — interactive dashboard and visualizations
- **Data Source** — HMRC Spending over £25,000 (Jul–Dec 2025), published at gov.uk

## Key Insights
- Total HMRC procurement spend: **£1.15 billion** across 6 months
- **Capgemini** is the top supplier at £222M, accounting for 19.4% of total spend
- **Top 3 suppliers** (Capgemini, Fujitsu, Accenture) represent ~37% of total spend — indicating high vendor concentration risk
- Spend peaked in **December 2025**

## Project Structure
- `hmrc_spend_analysis.sql` — SQL scripts for table creation, data cleaning, and analysis queries
